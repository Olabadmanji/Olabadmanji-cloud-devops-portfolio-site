# Event Announcement System (AWS)

## Overview
This project is a serverless event announcement platform built using AWS services.

## Architecture
- S3 for static frontend hosting
- API Gateway for REST endpoints
- AWS Lambda for backend logic
- SNS for email notifications

## Features
- Users can subscribe via email
- Event creation triggers email notifications
- Fully serverless and scalable

## AWS Services Used
- Amazon S3
- AWS Lambda
- Amazon SNS
- Amazon API Gateway
- IAM

## How It Works
1. Frontend is hosted on S3
2. Users interact via API Gateway
3. Lambda functions process requests
4. SNS sends notifications to subscribers

## Why This Project Matters
- Demonstrates event-driven architecture
- Shows serverless design principles
- Aligns with AWS Solutions Architect best practices
